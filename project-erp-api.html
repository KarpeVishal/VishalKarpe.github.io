ERP & Mobile App API

ASP.NET MVC 5 C# .NET JQuery JavaScript

Company: BillBox

Environment: ASP.NET MVC 5, C# .NET, JQuery, JavaScript

Project Description

Authored dynamic backend API layers for a large-scale ERP system and its companion multi-platform mobile application securely managing banking payment devices (sound boxes & QR codes) and facilitating remote field service operations.

Key Responsibilities & Features

  • Designed & implemented resilient REST APIs specifically for deep ERP integration bridging centralized device management, transparent transaction tracking, and instant status synchronizations.
  • Created specialized backend API endpoints to firmly support on-site scanning mechanisms, rigorous device validation logic, and precise activity logging.
  • Fortified all API endpoints with modern authentication and high-level encryption methodologies protecting consumer transaction metrics.
  • Generated extensive analytics covering service SLAs, device hardware performance over time, and user assignment history protocols.
